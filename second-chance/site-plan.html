Overview

Purpose

I am revamping a past version that I made, mostly focusing on more functionality and asethetic appeal.This website is for a nonprofit that my dad is going to startup, which functions as a way for children in the foster system to save up money for college. This is done by rebuilding and fixing up jeeps with a foster child, and once the jeep is fixed it is sold and used as scholarship money.

Audience

The audience of the website that I am focusing in on is those that could add to the non-profit through ways like donations and similar actions. For people that want to help in the organization, this website is what they will work/contact through and find info about past projects and the trajectory of Second Chance.

Dynamic elements

The first main dynamic element that I am adding is an interactive image carousel for past projects. Users will be able to flip through to see a main picture of a past project, then click to see more information on it. I'm changing the data to make it dynamic, so that new projects can be added to the website easily. If I have time I'm also going to add a forum for those reaching out to runners of Second Chance.
